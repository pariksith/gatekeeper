# GateKeeper

GateKeeper is a clean and minimal web portal designed to provide simple navigation and quick access to essential information. It offers a lightweight structure that can be extended into a personal site, registration system, or project dashboard.

## 🌟 Features

* **Minimal UI** – Focused on clarity and ease of use.
* **Quick Navigation** – Direct links to Home, Register, and Bio pages.
* **Contact Options** – Email and phone icons for instant communication.
* **Lightweight & Fast** – Static site design for speed and reliability.
* **Customizable** – Easy to extend and adapt for personal or project needs.

## 🚀 Getting Started

### View Online

Visit the live demo here:
👉 [GateKeeper on GitHub Pages](https://pariksith.github.io/gatekeeper/)

### Run Locally

1. Clone this repository:

   ```bash
   git clone https://github.com/pariksith/gatekeeper.git
   ```
2. Open the project folder and launch `index.html` in your browser.
